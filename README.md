# Конвертер систем счисления

## 📋 О проекте
Программа для конвертации чисел между системами счисления с графическим интерфейсом на Qt.

## Функционал
- Конвертация между **2, 8 и 10** системами счисления
- Поддержка прямого и обратного кода для двоичных чисел
- Валидация ввода и проверка на переполнение
- Копирование результата в буфер обмена
- Индикация статуса ввода

## Использование
1. Выберите исходную систему счисления (2, 8 или 10)
2. Выберите целевую систему счисления
3. Введите число
4. Нажмите кнопку "КОНВЕРТИРОВАТЬ"
5. При необходимости скопируйте результат

### Примеры
| Из | В | Ввод | Результат |
|----|---|------|-----------|
| 10 | 2 | 42 | 101010 |
| 2 | 10 | 1010 | 10 |
| 8 | 2 | 777 | 111111111 |

## Сборка
- Требуется Qt (версия 5 или 6)
- Откройте .pro файл в Qt Creator
- Соберите проект

## Технологии
- C++
- Qt Framework
- Windows API (для буфера обмена)
